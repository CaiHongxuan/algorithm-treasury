# 二叉树的最大深度

## 描述：
给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明: 叶子节点是指没有子节点的节点。

示例：
给定二叉树 [3,9,20,null,null,15,7]，

```
       3
      / \
     9  20
        / \
       15  7
```

返回它的最大深度 3 。

## 解答：

```php
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     public $val = null;
 *     public $left = null;
 *     public $right = null;
 *     function __construct($value) { $this->val = $value; }
 * }
 */
class Solution {

    /**
     * @param TreeNode $root
     * @return Integer
     */
    function maxDepth($root) {
        if ($root->val === null) {
            return 0;
        }
        $left = $right = 0;
        if ($root->left != null) {
            $left = $this->maxDepth($root->left);
        }
        if ($root->right != null) {
            $right = $this->maxDepth($root->right);
        }
        return 1 + ($left > $right ? $left : $right);
    }
}
```
